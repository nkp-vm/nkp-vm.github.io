timelinebox

Description

Add this attribute to make 'timeline' box element.

  • date-place: Date and place
  • title: Title of the item
  • image-src: Link to image
  • caption: caption text 
    <div timelinebox date-place="June 1970" title="test 1" image-src="views/content/img/timeline/..." caption="caption text">Detailed text about the item</div>

Usage

as attribute
<ANY timelinebox>
   ...
</ANY>
