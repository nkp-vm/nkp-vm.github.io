timeline-textbox

Description

Add this attribute to make 'timeline' box element.

  • type: 'quote' or 'regular'
  • date-place: Date and place
  • title: Title text
  • caption: Caption text
  • content: Content text'
  • bg-colour: background colour
  • bg-image: background image 
    <div timeline-textbox box-type="quote" bg-colour="#226666" bg-image="image1.jpg" date-place="June 1970" title-text="Sample title" caption-text="caption text"></div>

Usage

as attribute
<ANY timeline-textbox>
   ...
</ANY>
